# Images des produits

Placez vos images de produits dans ce dossier.

## Noms de fichiers recommandés

- `oignon.jpg` ou `oignon.png`
- `tomate.jpg` ou `tomate.png`
- `mil.jpg` ou `mil.png`
- `arachide.jpg` ou `arachide.png`
- `manioc.jpg` ou `manioc.png`
- `mangue.jpg` ou `mangue.png`
- `bissap.jpg` ou `bissap.png`
- `pasteque.jpg` ou `pasteque.png`

## Format recommandé

- Format : JPG ou PNG
- Taille : 400x400 pixels minimum
- Poids : < 500 KB par image pour un chargement rapide

Les images seront automatiquement utilisées si elles sont présentes dans ce dossier.
