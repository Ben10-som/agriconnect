# Guide d'utilisation en Wolof

## Fonctionnalité de traduction Wolof → Français

L'application supporte maintenant la publication vocale en **Wolof**, la langue la plus parlée au Sénégal.

## Comment utiliser

1. **Sélectionner la langue** : Choisissez "Wolof" avant de parler
2. **Parler en wolof** : Dites votre annonce en wolof
3. **Traduction automatique** : L'IA traduit automatiquement en français
4. **Extraction** : Les informations sont extraites et le formulaire est rempli

## Exemples de phrases en Wolof

### Exemple 1
> "Damaa jënd 50 sako soob ci 15000 FCFA. Nataalub tëlifon bi mooy 782840238"
> 
> Traduction : "Je vends 50 sacs d'oignons à 15000 FCFA. Mon numéro de téléphone est 782840238"

### Exemple 2
> "Am naa 100 kilo tamaat. Jëf bi mooy 500 FCFA. Tëlifon : 775555555"
> 
> Traduction : "J'ai 100 kilos de tomates. Le prix est 500 FCFA. Téléphone : 775555555"

### Exemple 3
> "Damaa jox 2 ton mbey ci 350000 junni. Nataal : 779876543"
> 
> Traduction : "Je vends 2 tonnes de mil à 350000 mille. Numéro : 779876543"

## Vocabulaire utile

### Produits
- **Sob** = Oignon
- **Tamaat** = Tomate
- **Mbey** = Mil
- **Gerte** = Arachide
- **Mbaxal** = Manioc
- **Mango** = Mangue
- **Bissap** = Bissap
- **Meloon** = Pastèque

### Unités
- **Sako** = Sac
- **Kilo** = Kilogramme
- **Ton** = Tonne

### Prix
- **Junni** = Mille (ex: "15 junni" = 15000)
- **FCFA** = FCFA

### Phrases utiles
- **Damaa jënd** = Je vends
- **Am naa** = J'ai
- **Jox naa** = Je donne
- **Jëf bi mooy** = Le prix est
- **Nataalub tëlifon** = Numéro de téléphone
- **Nataal** = Numéro

## Fonctionnement technique

1. **Reconnaissance vocale** : Le navigateur transcrit la voix (peut être approximatif pour le wolof)
2. **Traduction OpenAI** : Le texte wolof est traduit en français avec GPT
3. **Extraction IA** : Les informations sont extraites du texte français
4. **Sauvegarde** : Les données sont sauvegardées dans Firebase

## Notes importantes

- La reconnaissance vocale du navigateur peut ne pas être parfaite pour le wolof
- OpenAI traduit et extrait les informations avec une grande précision
- Si la traduction échoue, l'extraction essaie quand même avec le texte original
- Vous pouvez toujours corriger manuellement les informations après extraction
